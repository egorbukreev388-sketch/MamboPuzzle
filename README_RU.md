# Мамбо Puzzle — Android project

Это исходный код нативной Android-игры-головоломки в стиле «Мамбо».

Что уже есть:
- 1000 процедурных уровней.
- Размер поля растёт по этапам: 4×4, 6×6, 8×8, 10×10, 12×12.
- Математика уровня: равное количество двух типов элементов в каждой строке/столбце, запрет 3 одинаковых подряд, знаки `=` и `×` между соседними клетками.
- Экран игры, обучение, настройки, победный экран.
- Логика решения встроена через заранее сгенерированное корректное решение уровня; подсказка заполняет одну правильную клетку.

Как собрать APK:
1. Установи Android Studio.
2. Открой папку `MamboPuzzle` как проект.
3. Дождись синхронизации Gradle.
4. Build → Build Bundle(s) / APK(s) → Build APK(s).
5. APK будет в `app/build/outputs/apk/debug/app-debug.apk`.

Через терминал, если Gradle/Android SDK уже установлены:
```bash
cd MamboPuzzle
gradle :app:assembleDebug
```
